# **App Name**: Workout Tracker

## Core Features:

- Customizable Exercise Library: Allow users to specify and name exercises
- Set Tracking: A system to record sets, reps, and weight for each exercise during a workout.
- Workout Display: Display current workout exercises
- Real-Time Logging: Log sets in real time

## Style Guidelines:

- Primary color: Saturated teal (#4db6ac) to suggest fitness and health.
- Background color: Very light grayish-teal (#e0f2f1) for a clean, unobtrusive background.
- Accent color: Analogous dark cyan (#4dd0e1) to indicate active components or highlights, suggesting speed and power.
- Body and headline font: 'Inter' sans-serif, to give a clean, modern, objective feel.
- Use clear, simple icons to represent different muscle groups and exercises.
- Clean and spacious layout with clear sections for exercise selection, set logging, and workout summary.
- Subtle transitions and animations to provide feedback when logging sets or completing exercises.
